| Former Location | New Location |
|-----------------|-------------|
| `github-api-adapter.md` | [examples/adapters/github-api-adapter.md](https://github.com/MCPAQL/examples/blob/develop/adapters/github-api-adapter.md) |

## Why the Move

Example adapters are non-normative content. Keeping them in a dedicated repository:

- Separates normative specifications from illustrative examples
- Allows examples to evolve independently of the protocol spec
- Makes it easier for contributors to add new examples
