# Licensing Scope (Spec Repo)

This repository uses **split licensing**:

- **Documentation / prose**: Creative Commons Attribution 4.0 International (**CC BY 4.0**) — see `LICENSE-DOCS`.
- **Software artifacts**: GNU Affero General Public License v3.0 (**AGPL-3.0**) — see `LICENSE`.

If you are unsure which license applies to a file, use the path-based rules below.

## CC BY 4.0 applies to

Documentation and community text:

- `spec/docs/`
- `spec/README.md`
- `spec/CHANGELOG.md`
- `spec/CODE_OF_CONDUCT.md`
- `spec/SECURITY.md`
- `spec/CONTRIBUTING.md`
- `spec/.github/`

## AGPL-3.0 applies to

Artifacts intended to be consumed by tooling/implementations:

- `spec/schemas/`
- `spec/tests/`
- `spec/SPEC_VERSION`
- Any source code or scripts added later (for example: `spec/src/`, `spec/scripts/`, `spec/tools/`)
