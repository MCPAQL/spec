# Trademarks

This document describes trademark and attribution guidelines for MCP-AQL.

## Marks

The names “MCP-AQL”, “MCPAQL”, “Model Context Protocol – Advanced Agent API Adapter Query Language”, and associated logos (if any) are claimed as trademarks of Mick Darling.

## Compatibility and naming

You may use the marks to truthfully describe compatibility with MCP-AQL (for example, “compatible with MCP-AQL”) provided that:

- You do not imply endorsement, sponsorship, or affiliation without written permission.
- You include attribution to Mick Darling in documentation when practical.
- You do not use the marks as the primary name of a forked/rebranded product in a way that is likely to cause confusion as to origin.

## Attribution statement (recommended)

Recommended attribution language:
